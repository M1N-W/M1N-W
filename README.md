> **High School Student · Aspiring Software Engineer · Builder**

A high school student from Benjamarachutit Ratchaburi School with a strong passion for technology. I ship projects first, then learn the internals.

### 💡 My Coding Journey & Mindset
I started my journey through "Vibe-Coding," utilizing AI tools to bring my ideas to life. This experience sparked my drive to dive deeper. I realized that if I could build functional projects with AI, mastering the core code would make me unstoppable! 

So far, I've built a wave-survival game engine, a production LINE bot with 40+ real users, and a healthcare assistant bot — all before finishing high school. Now, I'm transitioning from AI-assisted development to full-stack engineering by building a solid foundation in core computer science concepts.

- 🌍 **Based in:** Ratchaburi, Thailand

- 🤝 **Open to collaborating on:** Web games, student-facing tools, open-source projects, and anything that ships to real users
